LSTM简介

  • LSTM(Long Short-Term Memory Networks,长短时记忆网络)，由Hochreiter和Schmidhuber于1997年提出，目的是解决一般循环神经网络中存在的梯度爆炸（输入信息激活后权重过小）及梯度消失（例如sigmoid、tanh的激活值在输入很大时其梯度趋于零）问题，主要通过引入门和Cell状态的概念来实现梯度的调整，已被大量应用于时间序列预测等深度学习领域。

+　早在1997年，LSTM在论文 《LONG SHORT-TERM MEMORY 》中被提出，直到2014年才进入高速发展阶段。它们属于循环神经网络家族- RNN，以及门控循环单元GRU。
+　LSTM有两个缺点：首先LSTM不容易在训练过程中并行化；其次由于它们具有周期性，它们可以建模的序列长度是有限制的。

LSTM算法流程

LSTM采用了门控输出的方式，即三门（输入门、遗忘门、输出门）两态（Cell State长时、Hidden State短时）。结构图如下所示（图源Understanding LSTM Networks）。作为机器学习算法的一种，LSTM 的预测过程也分为正向传播和反向传播两个过程。

正向传播过程

隐藏层

如上图所示，一个LSTM单元应当包含的权值参数有：

  • WfhW_{fh}Wfh​：遗忘门与短时记忆的权值向量。
  • WfxW_{fx}Wfx​：遗忘门与特征输入的权值向量。
  • bfb_{f}bf​：遗忘门的偏置。
  • WihW_{ih}Wih​：输入门与短时记忆的权值向量。
  • WixW_{ix}Wix​：输入门与特征输入的权值向量。
  • bib_{i}bi​：输入门的偏置
  • WchW_{ch}Wch​：产生新记忆信息部分与短时记忆的权值向量。
  • WcxW_{cx}Wcx​：产生新记忆信息部分与特征输入的权值向量。
  • bcb_{c}bc​：产生新记忆信息部分的偏置
  • WohW_{oh}Woh​：输出门与短时记忆的权值向量。
  • WoxW_{ox}Wox​：输出门与特征输入的权值向量。

设：

  • 序列总长度为：nnn，t=1,2,...,n−1,nt=1,2,...,n-1,nt=1,2,...,n−1,n。
  • ttt时刻的特征输入向量：xtx_{t}xt​。
  • t−1t-1t−1时刻的短时记忆（t−1t-1t−1时刻LSTM单元的输出值）：ht−1h_{t-1}ht−1​。

遗忘门

遗忘门接收ht−1h_{t-1}ht−1​和xtx_{t}xt​，得到要舍弃的信息ftf_{t}ft​，公式为：

Ft=Wfhht−1+Wfxxt+bf(1)F_{t}=W_{fh}h_{t-1}+W_{fx}x_{t}+b_{f} \tag{1} Ft​=Wfh​ht−1​+Wfx​xt​+bf​(1)

ft=σ(Ft)(2)f_{t}=\sigma(F_{t}) \tag{2} ft​=σ(Ft​)(2)

输入门

输入门可分为两个部分：输入门部分和产生新记忆信息部分。

  • 接收ht−1h_{t-1}ht−1​和xtx_{t}xt​，得到要更新的信息iti_{t}it​，公式为：

It=Wihht−1+Wixxt+bi(3)I_{t}=W_{ih}h_{t-1}+W_{ix}x_{t}+b_{i} \tag{3} It​=Wih​ht−1​+Wix​xt​+bi​(3)

it=σ(It)(4)i_{t}=\sigma(I_{t}) \tag{4} it​=σ(It​)(4)

  • 以及得到新的记忆信息c~t\widetilde{c}_{t}ct​（这里的c~t\widetilde{c}_{t}ct​代表的是图中的C~t\widetilde{C}_{t}Ct​），公式为：

C~t=Wchht−1+Wcxxt+bc(5)\widetilde{C}_{t}=W_{ch}h_{t-1}+W_{cx}x_{t}+b_{c} \tag{5} Ct​=Wch​ht−1​+Wcx​xt​+bc​(5)

c~t=tanh⁡(C~t)(6)\widetilde{c}_{t}=\tanh(\widetilde{C}_{t}) \tag{6} ct​=tanh(Ct​)(6)

长时记忆部分

由遗忘门和记忆门的信息可以得到ttt时刻的（长时记忆）状态值，公式为：

Ct=Ct−1ft+itc~t(7){C}_{t}={C}_{t-1}f_{t}+i_{t}\widetilde{c}_{t} \tag{7} Ct​=Ct−1​ft​+it​ct​(7)

输出门

输出门接收ht−1h_{t-1}ht−1​和xtx_{t}xt​，得到要输出的信息（这是一个中间版本）oto_{t}ot​，公式为：

Ot=Wohht−1+Woxxt+bo(8)O_{t}=W_{oh}h_{t-1}+W_{ox}x_{t}+b_{o} \tag{8} Ot​=Woh​ht−1​+Wox​xt​+bo​(8)

ot=σ(Ot)(9)o_{t}=\sigma(O_{t}) \tag{9} ot​=σ(Ot​)(9)

短时记忆部分

并再根据长时记忆值得到最终LSTM单元的输出值（短时记忆），公式为：

ht=tanh(Ct)∗ot(10)h_{t}=tanh(C_{t})\ast o_{t} \tag{10} ht​=tanh(Ct​)∗ot​(10)

输出层

如果LSTM隐层与输出层之间是全连接的，并且采用sigmoid激活函数，得到预测输出y^t\hat{y}_{t}y^​t​，那么公式为：

Y^t=Woutht+bout(11)\hat{Y}_{t}=W_{out}h_{t}+b_{out} \tag{11} Y^t​=Wout​ht​+bout​(11)

y^t=σ(Y^t)(12)\hat{y}_{t}=\sigma(\hat{Y}_{t}) \tag{12} y^​t​=σ(Y^t​)(12)

损失函数

损失函数用来评价模型的预测值和真实值不一样的程度。这里使用MSE(Mean Square Error, 均方差)来进行评估评估实际输出与期望输出的误差。MSE的公式为：（y^t\hat{y}_{t}y^​t​为预测值，yty_{t}yt​标签值，nnn为序列长度）

E=1n∑t=1n(y^t−yt)2(13)E=\frac{1}{n}\sum_{t=1}^{n}(\hat{y}_t-y_{t})^{2} \tag{13} E=n1​t=1∑n​(y^​t​−yt​)2(13)

反向传播过程

反向传播就是根据正向传播的误差来求梯度，然后根据梯度调整原来的权重。

  • sigmoid函数的导数为：σ′(x)=σ(x)∗(1−σ(x)){\sigma}'(x)=\sigma(x) \ast (1-\sigma(x))σ′(x)=σ(x)∗(1−σ(x))。
  • tanh函数的导数为：tanh′(x)=1−tanh2(x){tanh}'(x)=1-tanh^{2}(x)tanh′(x)=1−tanh2(x)。
  • 使用grad(x)grad(x)grad(x)来代表误差EEE关于xxx的梯度。
  • 对于最后一个时刻t=nt=nt=n，需要参与计算的Cn+1C_{n+1}Cn+1​，fn+1f_{n+1}fn+1​，in+1i_{n+1}in+1​，c^n+1\hat{c}_{n+1}c^n+1​，on+1o_{n+1}on+1​都为0。

grad(Y^t)=∂E∂Y^t=∂y^t∂Y^t∗∂E∂y^t=σ′(x)∗grad(y^t)(15)grad(\hat{Y}_{t})=\frac{\partial{E}}{\partial{\hat{Y}}_{t}}=\frac{\partial{\hat{y}}_{t}}{\partial{\hat{Y}}_{t}}\ast\frac{\partial{E}}{\partial{\hat{y}}_{t}}={\sigma}'(x)\ast grad(\hat{y}_{t}) \tag{15} grad(Y^t​)=∂Y^t​∂E​=∂Y^t​∂y^​t​​∗∂y^​t​∂E​=σ′(x)∗grad(y^​t​)(15)

对隐层与输出层之间的连接权值的梯度计算：

grad(Wout)=∂E∂Wout=∂Y^t∂Wout∗∂E∂Y^t=ht∗grad(Y^t)(16)grad(W_{out})=\frac{\partial{E}}{\partial{W}_{out}}=\frac{\partial{\hat{Y}}_{t}}{\partial{W}_{out}}\ast\frac{\partial{E}}{\partial{\hat{Y}}_{t}}=h_{t}\ast grad(\hat{Y}_{t}) \tag{16} grad(Wout​)=∂Wout​∂E​=∂Wout​∂Y^t​​∗∂Y^t​∂E​=ht​∗grad(Y^t​)(16)

grad(bout)=∂E∂bout=∂Y^t∂bout∗∂E∂Y^t=grad(Y^t)(17)grad(b_{out})=\frac{\partial{E}}{\partial{b}_{out}}=\frac{\partial{\hat{Y}}_{t}}{\partial{b}_{out}}\ast\frac{\partial{E}}{\partial{\hat{Y}}_{t}}=grad(\hat{Y}_{t}) \tag{17} grad(bout​)=∂bout​∂E​=∂bout​∂Y^t​​∗∂Y^t​∂E​=grad(Y^t​)(17)

优化算法

这里采用的是Adam（Adaptive Moment Estimation，自适应矩估计）算法。

设在ttt时刻下：

  • wtw_{t}wt​为权重，gtg_{t}gt​为wtw_{t}wt​的梯度，mtm_{t}mt​为wtw_{t}wt​的一阶矩估计，rtr_{t}rt​为wtw_{t}wt​的二阶矩估计。
  • α\alphaα为学习率，β1\beta_{1}β1​和β2\beta_{2}β2​为控制指数衰减的常数，ϵ\epsilonϵ为用于数值稳定的常数。
  • α\alphaα一般需要调节，其他均可采用默认设置：β1=0.9\beta_{1}=0.9β1​=0.9，β2=0.999\beta_{2}=0.999β2​=0.999，ϵ=10−8\epsilon=10^{-8}ϵ=10−8。

算法流程如下：

  1. 更新一阶矩估计mtm_{t}mt​：mt=β1∗mt−1+(1−β1)∗gtm_{t}=\beta_{1} \ast m_{t-1}+(1-\beta_{1}) \ast g_{t}mt​=β1​∗mt−1​+(1−β1​)∗gt​

  2. 更新二阶矩估计rtr_{t}rt​：rt=β2∗rt−1+(1−β2)∗gt2r_{t}=\beta_{2} \ast r_{t-1}+(1-\beta_{2}) \ast g_{t}^{2}rt​=β2​∗rt−1​+(1−β2​)∗gt2​

  3. 求得mtm_{t}mt​的偏差修正m^t\hat{m}_{t}m^t​：m^t=mt1−β1\hat{m}_{t}=\frac{m_{t}}{1-\beta_{1}}m^t​=1−β1​mt​​

  4. 求得rtr_{t}rt​的偏差修正r^t\hat{r}_{t}r^t​：r^t=rt1−β2\hat{r}_{t}=\frac{r_{t}}{1-\beta_{2}}r^t​=1−β2​rt​​

  5. 更新权重wtw_{t}wt​：wt=wt−1−α∗m^tr^t+ϵw_{t}=w_{t-1}-\alpha \ast \frac{\hat{m}_{t}}{\sqrt{\hat{r}_{t}}+\epsilon}wt​=wt−1​−α∗r^t​​+ϵm^t​​

注：这里的函数形参采用指针形式传入地址，以修改值。

构建LSTM网络

  • 定义全局的二维数组作为权值矩阵，并在开始时分配内存，在结束时释放。
  • 由优化算法可知，每个权值都有对应的一阶矩估计和二阶矩估计，因此需要定义的参数除了上述算法中的(12+2)(12+2)(12+2)个之外，还需要(12+2)∗2(12+2)*2(12+2)∗2个参数，总共14∗3=4214*3=4214∗3=42个参数。
  • 不足之处：为了更好的可扩展性和可读性，事实上应当定义一个LSTM单元为结构体。
  • 网络建立后就需要对权值进行初始化，这里采用正太分布初始化权重，一、二阶矩估计和偏置初始化为0。
